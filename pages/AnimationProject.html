The Monster

Context

Back during my 1st semester, I took animation as a complementary course, thinking it might help me with animations in games. Unfortunately, it didn't help much, but it was still pretty fun, if not time consuming.

Description

A basic stop-motion animation I did with a colleague using play-doh, paper and a camera. I often had to stay late after class to reach a point at which I could stop, along with not having enough time in general. All things considered, it turned out pretty well.
